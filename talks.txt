7|An all too short introduction to JVM performance|Java, JVM Ytelse, JIT|Rafael Winterhalter|While writing code for a virtual machine can simplify our life, considering an abstract machine also makes it often difficult to reason about runtime performance. For this reason, many developers consider code that was written for a physical machine to run more performant. To the contrary, modern JVMs often outperform native programs. This presentation sneaks a peek at the inner life of the HotSpot virtual machine, its just-in-time compiler and its interplay with a computer's hardware. At the same time, it sets some ground rules that lead on to performant Java code.
